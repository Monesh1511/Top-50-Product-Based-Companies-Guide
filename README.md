# TechCareerCompass

An interactive career dashboard for software engineering freshers to explore top product-based companies, compare interview difficulty, understand salary ranges, and follow a practical preparation roadmap.

## Preview

TechCareerCompass is a single-page HTML project that helps freshers quickly compare product-based companies by:

- Interview difficulty
- DSA level
- System design expectations
- Fresher hiring frequency
- India entry-level salary range
- Key technical skills
- FAANG+ preparation targets
- LeetCode-based roadmap

## Why This Project?

Freshers often know they want to join a product-based company, but they do not always know where to start. This dashboard turns scattered interview and company information into a clean, searchable, filterable guide.

## Features

- Interactive tabs for company list, tier rankings, FAANG+, fresher-friendly companies, and roadmap
- Search box to quickly find companies by name or industry
- Filters for difficulty and fresher hiring frequency
- Clean salary range indicators
- Visual badges for difficulty levels
- Product-company-focused list of 50 companies
- Responsive design for desktop and mobile
- Single HTML file with CSS and JavaScript included

## Project Sections

| Section | What It Shows |
| --- | --- |
| All 50 Companies | Complete searchable company table |
| Tier Rankings | Companies grouped by interview difficulty |
| FAANG+ | Elite product companies and salary comparison |
| Freshers-Friendly | Companies with better fresher hiring opportunities |
| Prep Roadmap | DSA and system design roadmap based on target level |

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js for visual comparison

## How To Run

1. Download or clone this repository.
2. Open `top50_product_companies_guide_styled.html` in any browser.
3. Use the tabs, search box, and filters to explore the dashboard.

No installation, build step, or server is required.
